 Sales Data Analysis – Vrinda Store

Project Overview
This project focuses on analyzing sales data of Vrinda Store using Microsoft Excel. The objective was to clean the raw dataset, analyze sales performance, and create an interactive dashboard to derive meaningful business insights.

 🛠️ Tools Used
- Microsoft Excel

 Key Tasks Performed
- Cleaned raw data by removing duplicates and handling missing values
- Standardized and organized sales data for analysis
- Used Pivot Tables to summarize sales performance
- Applied Excel formulas for calculations and data transformation
- Created interactive dashboards using charts and slicers

 Key Insights
- Identified top-selling products contributing the highest revenue
- Analyzed monthly sales trends and seasonal patterns
- Found highest sales contributing categories
- Compared performance across different regions/customer segments

 Objective
To analyze sales data and provide actionable insights that can help improve business decision-making for Vrinda Store.

 Outcome
Successfully developed a dynamic Excel dashboard that provides a clear view of sales performance, trends, and key business metrics.
